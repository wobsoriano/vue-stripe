## FAQs

::accordion

  :::accordion-item{label="Is Vue Stripe official?"}
  Vue Stripe is not an official Stripe library. It's a community-maintained project that provides 1:1 parity with the React Stripe.js SDK. For an official Stripe Partner solution, consider using the [Stripe Partner Vue SDK](https://vuestripe.com).
  :::

  :::accordion-item{label="Can I use React Stripe.js documentation?"}
  Yes! Vue Stripe provides 1:1 parity with `@stripe/react-stripe-js`. You can follow the [React Stripe.js documentation](https://docs.stripe.com/stripe-js/react) and adapt the examples to Vue syntax.
  :::

  :::accordion-item{label="What's the difference between Elements and CheckoutProvider?"}
  - **Elements**: For flexible payment collection with full control over the UI
  - **CheckoutProvider**: For Elements with Checkout Sessions (Embedded Checkout)
  :::

  :::accordion-item{label="Do I need to use both Elements and CheckoutProvider?"}
  No. Use `Elements` for standard payment flows, or `CheckoutProvider` for Checkout Session integrations.
  :::

  :::accordion-item{label="How do I handle errors?"}
  Elements emit `@error` events, and Stripe methods return error objects:

  ```vue
  <script setup>
  const stripe = useStripe()
  const elements = useElements()

  async function handleSubmit() {
    const { error } = await elements.value.submit()
    if (error) {
      console.error(error.message)
    }
  }
  </script>
  ```
  :::

  :::accordion-item{label="Can I use Vue Stripe with SSR?"}
  Yes. Pass `null` initially for the `stripe` prop and update it when Stripe loads:

  ```vue
  <script setup>
  const stripePromise = ref(null)

  onMounted(() => {
    stripePromise.value = loadStripe('pk_test_xxx')
  })
  </script>
  ```
  :::

  :::accordion-item{label="How do I style Elements?"}
  Use the Appearance API in the `Elements` provider:

  ```vue
  <Elements
    :stripe="stripePromise"
    :options="{
      appearance: {
        theme: 'stripe',
        variables: {
          colorPrimary: '#0570de',
        },
      },
    }"
  >
    <CheckoutForm />
  </Elements>
  ```
  :::

  :::accordion-item{label="Can I use multiple Elements?"}
  Yes, you can use multiple Elements in the same `Elements` provider, but only one of each type.
  :::

  :::accordion-item{label="How do I access the underlying Stripe element?"}
  Use `getElement()`:

  ```vue
  <script setup>
  const elements = useElements()
  const paymentElement = elements.value?.getElement(PaymentElement)
  </script>
  ```
  :::

  :::accordion-item{label="Is TypeScript supported?"}
  Yes, Vue Stripe is written in TypeScript and includes full type definitions.
  :::

::
